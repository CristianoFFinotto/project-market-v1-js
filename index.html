Expiry List

A javascript program that outputs a list of supermarket goods filtered by expiry date

This project allows you to control the expiry dates and the checks carried out. These will be divided according to the week of arrival. In addition, depending on the expiration date and shelf life, they will be categorized as "new", "old" or "expired", with different colors.

Press F12 to run the program.
